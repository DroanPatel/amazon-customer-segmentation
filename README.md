# Amazon Customer Segmentation Analysis

## Project Overview
This project analyzes Amazon customer transaction data using Python, SQL, Power BI, and Machine Learning techniques to identify customer purchasing behavior, high-value customer groups, and sales trends.

## Tools & Technologies
- Python
- SQL
- Power BI
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- K-Means Clustering

## Dataset
The dataset contains 50,000+ Amazon customer transaction records including:
- Customer IDs
- Purchase frequency
- Product categories
- Revenue
- Order dates
- Customer spending behavior

## Project Objectives
- Perform customer segmentation using RFM analysis
- Identify high-value and loyal customers
- Analyze purchasing behavior and sales trends
- Build dashboards for business insights

## Key Insights
- High-value customers contributed over 38% of total revenue while representing less than 20% of the customer base
- Loyal customers showed significantly higher repeat purchase behavior
- Seasonal sales spikes were observed during holiday shopping periods
- Certain product categories generated consistently higher customer retention rates

## Dashboard Features
- Customer segmentation overview
- Revenue analysis
- Regional sales trends
- Repeat customer behavior
- RFM analysis visualizations

## Project Outcome
The project helped identify high-value customer groups and provided data-driven recommendations for customer retention and targeted marketing strategies.
